레더 규칙과 진행 방식

  • 모든 순위는 에테르 숫자로 정해 집니다
  • 모든 플레이는 솔로 플레이여야 합니다 (파티 X)
  • 비약, 향 사용 허용 됩니다
  • 순위는 정해진 시간에 일괄 적용 됩니다
  • 1등 갱신될 때마다 해당 영상은 유튜브에 공개됩니다

상품 목록

  • 전체 1등 : 10만원 상품권
  • 5,6,7,8단계별 1등 : 5만원 상품권
  • 직업별 1등 : 5만원 상품권

참가 방법

  • PoEasy 디스코드 클릭
  • Dia4 Ladder 채널 클릭
  • 본인 영상 업로드
